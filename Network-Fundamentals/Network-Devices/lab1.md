# This Lab was made to show and understand what each device does in a real world network topology using Cisco Packet Tracer.
 The Network Topologies in this lab is the tokyo branch and the New York branch connected over the internet.

 # Devices used in each topology

##New York Branch
2x PCs -end user client devices
1x Cisco 2960 switch - connects PCs in local LAN
1x 2911 router - routes traffic between network
1x 5505 firewall - Firewall controlling traffic in/out 
  of the New York network

 ## INTERNET DEVICE

  Device is a 2911 that connects the new york branch and the tokyo branch together

 ## Hacker device

  device is just a laptop does malicious attacks from this devices to both branches

## TOKYO Branch
2x servers - Provides service to al clients
1x Cisco 2960 switch - connects PCs in local LAN
1x 2911 router - routes traffic between network
1x 5505 firewall - Firewall controlling traffic in/out 
  of the Tokyo branch

 #  What I learned in this Lab
  - I Learned that switches connect within the same LAN
  - routers connect networks together
  - Firewalls sit between router and internet to filter traffic
